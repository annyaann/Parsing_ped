# Парсер карточек с pedsovet.org

## Описание
Программа парсит карточки с главной страницы образовательного портала pedsovet.org.

## Что извлекает
- Названия карточек 
- Ссылки на соответствующие статьи

## Источник данных
Источник данных - HTML-страница, она загружается с сайта: https://pedsovet.org/

 ## Инструкция по запуску

1. Склонируйте репозиторий
```
git clone https://github.com/annyaann/Parsing_ped
```
2. Перейдите в папку репозитория
```
cd pedagogika
```
3. Создайте виртуальное окружение
Windows:
```
python -m venv venv 
```
MacOs:
```
python3 -m venv venv 
```
4. Активируйте виртуальное окружение
Windows:
```
venv\Scripts\activate 
```
MacOs:
```
source venv/bin/activate 
```
5. Установите библиотеки из файла requirements.txt
```
pip install -r requirements.txt
```

## Запуск
Для запуска программы введите:
```
python parse_pedagogy.py
```
## Результат

- Данные выводятся в терминал в формате: `id) Название: Ссылка`
- Результаты сохраняются в файл `articles.json`
- JSON-файл содержит список словарей c ключами `Название`, `Ссылка`
